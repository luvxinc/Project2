# UI UX Pro Max 详细参考: UX 准则 + 交付前检查

> **加载时机**: 需要 UX 反模式检查 或 交付前自检时

## 1. UX 准则 (99 条, 按关键领域分组)

### 导航与信息架构 (10 条)
1. 主导航项 ≤ 7 个 (Miller's Law)
2. 当前位置始终可见 (面包屑/高亮)
3. 返回按钮行为可预测
4. 搜索框始终可见或 1-click 可达
5. 导航层级 ≤ 3 层
6. 移动端汉堡菜单有明确图标
7. 标签页文字描述功能而非抽象
8. 外部链接标注 ↗ 图标
9. 404 页面提供有用导航
10. 相关内容提供上下文链接

### 表单与输入 (15 条)
11. 每个输入有关联 label
12. 实时验证而非提交后
13. 错误消息说明如何修复
14. 密码有显示/隐藏切换
15. 自动聚焦到第一个可编辑字段
16. Tab 键按逻辑顺序
17. 长表单分步骤 (Wizard)
18. 提交按钮显示加载状态
19. 必填标 * 可选标 "(可选)"
20. 下拉 > 10 项时提供搜索
21. 日期选择器有手动输入选项
22. 文件上传显示进度
23. 表单部分完成可保存草稿
24. 成功提交有确认反馈
25. 危险操作有二次确认

### 反馈与状态 (10 条)
26. 每个操作都有视觉反馈 (≤100ms)
27. 加载状态有骨架屏或 Spinner
28. 空状态有引导文案和 CTA
29. 错误状态有重试选项
30. Toast 通知自动消失 (3-5s)
31. 进度条有百分比/步骤指示
32. 不可逆操作有警告
33. 成功状态有 ✓ 视觉确认
34. 网络断开有离线提示
35. 长任务有取消选项

### 可访问性 (WCAG 2.1 AA) (15 条)
36. 文字对比度 ≥ 4.5:1 (正文)
37. 大文字对比度 ≥ 3:1 (>18px)
38. 非文本元素对比度 ≥ 3:1
39. 所有图片有 alt 文字
40. 键盘可操作所有功能
41. 焦点指示器清晰可见
42. 尊重 prefers-reduced-motion
43. 尊重 prefers-color-scheme
44. 语义化 HTML (nav/main/article)
45. ARIA landmark 正确
46. 表格有 scope 和 caption
47. 视频有字幕
48. 不仅依赖颜色传达信息
49. 触摸目标 ≥ 44×44px
50. 文本可缩放至 200%

### 性能感知 (10 条)
51. 首屏渲染 ≤ 1.5s (LCP)
52. 交互延迟 ≤ 100ms (INP)
53. 布局偏移 ≤ 0.1 (CLS)
54. 图片懒加载 (viewport 外)
55. 关键 CSS 内联
56. 字体 font-display: swap
57. 预加载关键资源
58. 列表虚拟化 (>100 项)
59. 防抖搜索输入 (300ms)
60. 节流滚动事件

### 移动端优化 (10 条)
61. 触摸目标 ≥ 48px (Material 标准)
62. 底部导航 (≤5 项)
63. 下拉刷新
64. 手势导航 (左滑返回)
65. 横屏适配
66. 安全区域 (iPhone 刘海/底栏)
67. 输入键盘类型匹配 (email/tel/number)
68. 避免 Hover 依赖 (触屏无 hover)
69. 固定底部 CTA 按钮
70. 懒加载非首屏内容

## 2. 反模式清单 (必须避免)

| # | 反模式 | 问题 | 正确做法 |
|---|--------|------|---------|
| 1 | Emoji 做图标 | 不同系统渲染不同 | SVG: Heroicons/Lucide |
| 2 | 无 cursor-pointer | 用户不知道可点击 | 所有 clickable 加 cursor-pointer |
| 3 | 无过渡动画 | 界面感觉生硬 | transition 150-300ms |
| 4 | 纯黑文本 (#000) | 对比度太高, 刺眼 | 用 #1a1a1a 或 #0f172a |
| 5 | 纯白背景 (#fff) | 太亮, 缺少层次 | 用 #fafafa 或 #f8fafc |
| 6 | 无焦点样式 | 键盘用户看不到 | outline + focus-visible |
| 7 | 自动播放视频 | 用户反感 | 用户主动播放 |
| 8 | 弹窗无关闭 | 用户被困住 | ESC + 点击遮罩 + X 按钮 |
| 9 | 只用颜色区分 | 色盲无法区分 | 颜色+图标+文字 |
| 10 | 无限滚动无反馈 | 用户不知道要等 | "加载更多" 或 skeleton |

## 3. 交付前检查清单 (Pre-Delivery Checklist)

```
视觉:
[ ] 无 emoji 做图标 (用 SVG: Heroicons/Lucide)
[ ] 所有可点击元素有 cursor-pointer
[ ] 悬停有平滑过渡 (150-300ms)
[ ] 亮色模式文字对比度 ≥ 4.5:1
[ ] 暗色模式文字对比度 ≥ 4.5:1

交互:
[ ] 键盘导航焦点状态可见
[ ] ESC 关闭所有浮层/弹窗
[ ] Loading 状态有骨架屏
[ ] Error 状态有重试选项
[ ] Empty 状态有引导文案

响应式:
[ ] 375px (iPhone SE)
[ ] 768px (iPad 竖屏)
[ ] 1024px (iPad 横屏)
[ ] 1440px (桌面)
[ ] 尊重 prefers-reduced-motion
[ ] 尊重 prefers-color-scheme

性能:
[ ] 图片懒加载
[ ] 列表虚拟化 (如果 >100 项)
[ ] 搜索防抖 (300ms)
[ ] 无 console.log
```
