# Workflows 真相源索引

> 本页是流程入口。角色技能文件只引用流程，不重复状态机与门禁细则。

## 主流程

- `build.md` — 新建/重构/功能开发闭环
- `ship.md` — 本地开发/CI-CD/部署
- `guard.md` — TDD/审查/排查/事故响应
- `ui.md` — UI/主题/动画
- `learn.md` — 学习/更新 GitHub 库并纳入工具库（分类/切片/去重）

## 使用约定

1. 流程编排在 `workflows/*.md`
2. 角色职责在 `skills/*.md`
3. 规则细则在 `rules/*.md`
4. 三层禁止重复维护同一条款
