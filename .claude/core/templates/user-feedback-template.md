# 用户反馈记录

> 路径: `.claude/projects/{project}/data/progress/user-feedback.md`
> 用途: 记录用户的偏好、不满、期望，供后续需求采集参考

---

## 反馈记录

| 日期 | 功能/模块 | 反馈类型 | 反馈内容 | 处理状态 |
|------|---------|---------|---------|---------|
| YYYY-MM-DD | {模块名} | 喜好 / 不满 / 建议 | {用户原话或提炼} | ⬜ 待处理 / ✅ 已处理 |

---

## 反馈类型说明

| 类型 | 说明 | 写入哪里 |
|------|------|---------|
| **偏好/风格** | UI 颜色、布局、交互偏好 | PROJECT-MEMORY.md (UIUX 分类) |
| **功能不满** | 某功能不好用或有 Bug | ERROR-BOOK.md |
| **功能建议** | 希望增加新功能 | 此文件 → 下次需求采集时参考 |
| **数据口径** | 某字段的计算方式/含义 | PROJECT-MEMORY.md (数据口径分类) |
